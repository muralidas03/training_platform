# Training Platform

## Features
- Course CRUD
- Student Registration/Profile
- Training Schedule CRUD
- Student Opt-in/Out
- REST APIs with Auth

## Tech
- Django
- Django REST Framework
- SQLLITE3

## Setup
- pip install -r requirements.txt
- python manage.py migrate
- python manage.py createsuperuser
- python manage.py runserver



## Unit Tests
- python manage.py runserver
